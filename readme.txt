Primeiro instale o npm junto com o node.js para conseguir rodar a aplicação. Link para baixar o npm: https://www.npmjs.com/get-npm?utm_source=house&utm_medium=homepage&utm_campaign=free%20orgs&utm_term=Install%20npm.

Depois abrir a pasta do projeto no cmd (windows + r , digite cmd e aperte enter), e rode o comando npm install na raiz do projeto (pasta VidaLink)
Obs: esse passo pode ser feito no visual studio code ou sublime também.

Após instalar todas as dependencias, usar o comando ng serve para ele executar a aplicação, depois que ele terminar de executar todas as tarefas ele vai pedir para abrir o navegador e digitar http://localhost:4200
na barra do de navegação( o numero 4200 é a porta que ele usa PODE SER QUE SEJA OUTRA, se atentar a qual ele vai pedir).

Nome de usuário para acessar o sistema: vidalink
Senha para acessar o sistema: vida2018
